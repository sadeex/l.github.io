Firebase Setup Required

Connect Fire Database to your Firebase project in 6 easy steps.

Step 1 — Go to console.firebase.google.com → Create project
Step 2 — Authentication → Get started → Email/Password → Enable
Step 3 — Firestore Database → Create database → Test mode
Step 4 — Storage → Get started → Test mode (for file uploads)
Step 5 — Project Settings → Web App → Register → Copy firebaseConfig
Step 6 — Open fire-database-v2.html → Find firebaseConfig at top → Paste your values → Save & Reload
Firestore Rules (Paste in Rules tab): 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /posts/{id}  { allow read: if true; allow write: if request.auth != null; }
    match /users/{id}  { allow read: if request.auth != null; allow write: if request.auth.uid == id; }
  }
}
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} { allow read: if true; allow write: if request.auth != null; }
  }
}
